SK Planet Weather API (https://developers.skplanetx.com)가 T Developers (https://developers.sktelecom.com/content/sktApi/view/?svcId=10113)로 통합됨에 따라 이 프로젝트에 있는 API는 이용이 불가합니다. 또한 GoogleMap API를 사용중인데 구글에서는 API를 무료 3번 유로는 결제해서 사용 해야하는식으로 API 사용 정책을 변경하였기에 해당 소스코드는 작동은 되나 구글 (https://cloud.google.com/maps-platform/?hl=ko)에 가셔서 결제를 하셔야 원할하게 사용 가능할겁니다.
 사실상 이 프로젝트는 손대기에는 API 전체 소스코드를 갈아엎어야해서 특별히 필요해지지 않는이상 저는 손대지 않을 예정이며 무료 지도 API 활용 소스코드를 확인하시려면 SPC(SchoolPerformanceCalculator) 프로젝트를 확인하시길 바랍니다.
 해당 프로젝트에서는 현재 Daum API를 통한 경도, 위도 구하기와 Naver Papago API를 통한 번역의 사용 예제를 구경할 수 있습니다.
